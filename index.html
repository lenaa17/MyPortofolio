Latest Project

Lost and Found

This team project, Lost n Found, was developed during a bootcamp as a platform to help users find lost items. The website enables users to upload pots about items they've found. facilitating the process of reuniting owners with their missing belongings. Built using laravel, Lost and Found steamlines the search for lost possesions

Ecourt

The E-court team project was developed for a Software Engineering course using PHP, Laravel and Figma. E-court is a web-based application that helps users find service providers or technicians specializing in electronics. The tasks included designing the prototype, which covered the homepage, a page explaining how the website works, and the About Us page, as well as contributing to the front-end development of the Consultation page.

Intertip

This project is from the Artificial Intelligence course, focusing on creating a prototype for an application called Intertip. The Intertip app allows users to book tickets and features a unique "shared account" option that sets it apart from other applications. The prototype of this app was created using Figma.

WatcHGoods

For the final project in the Human-Computer Interaction course, a prototype for the WatcHGoods website was created using Figma, followed by implementation with HTML, CSS, and JavaScript. WatcHGoods is an online platform that sells pre-owned luxury watches authenticated by experts. The project involved designing and developing the homepage, products page, About Us, and Contact Us pages.

SOUNDCLOUD

This project is part of the Human-Computer Interaction course, focusing on creating a prototype and adding necessary features for the SoundCloud app. SoundCloud is an audio streaming platform that allows users to upload, listen to, and share music and podcasts. The app's prototype was designed using Figma. Additionally, the project involved adding new premium features, namely "NextPro," "NextFamily," "NextMini," and "NextPlus," each offering different benefits.

CALM

The midterm exam project for the Human-Computer Interaction course involved designing a prototype for a mental health app called Calm using Figma, with a one-week deadline. The screens created included the splash screen, login, sign-up, profile, homepage, service selection, consultation, payment, chat with a psychologist, and consultation history. This project provided a deeper understanding of Figma and its tools.
